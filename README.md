# MuseHub

MuseHub is a React + Vite + TypeScript application prepared for AI-assisted development. The UI system is HeroUI v3 with Tailwind CSS v4 and local CSS for art direction.

## Commands

```bash
npm run dev
npm run lint
npm run format:check
npm run test
npm run build
```

## Project Structure

- `src/app`: app assembly, navigation, route contracts, global config.
- `src/pages`: route-level page contracts and future page implementations.
- `src/widgets`: large page regions.
- `src/features`: user-facing workflows.
- `src/entities`: domain models and entity contracts.
- `src/shared`: reusable UI, contracts, and primitives.
- `src/styles`: global styles, HeroUI import, theme tokens, responsive rules.
- `docs`: project documentation and architectural decision records.
